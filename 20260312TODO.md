## Inngest 파이프라인 체인 (참고)

```
property/comp-set.build
  → scrape-own-listing (자사 리스팅 상세 + 캘린더)
  → property/comp-set.build-members
    → build-comp-set (비교군 구성)
    → property/comp-set.scrape-batch
      → scrape-comp-details (비교군 상세 스크래핑)
      → property/market-intel.analyze
        → analyze-market-intel (시장 분석)
        → property/diagnosis.run
          → run-diagnosis (진단 → status: 'active')
          → property/strategy.generate
            → generate-strategy (AI 전략 + 가격 + 콘텐츠 + 액션)
```
